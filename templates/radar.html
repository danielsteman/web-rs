Radar
This radar was open-sourced by Zalando. It seemed like a good way to visualize the tools I'm currently using or have used in the past. I decided on these four quadrants because they best represent the clusters of technologies found in my projects. Adopt is the label for technologies I'm actively using and I understand well. Trial is the label for technologies I'm using in a project to give them a shot but don't feel comfortable enough with to the extend that it would be my go-to tool in a next project. Assess bundles technologies that seem promising and worth investing some research in. Hold contains technologies I have used in the path but wouldn't choose again because I have found a better alternative, for my use cases at least.
